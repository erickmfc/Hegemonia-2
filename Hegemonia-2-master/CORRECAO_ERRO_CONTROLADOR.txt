CAUSA:
------
O objeto obj_controlador_construcao foi criado programaticamente mas não foi 
devidamente registrado no sistema do GameMaker Studio. Embora os arquivos tenham 
sido adicionados aos arquivos .yyp e .resource_order, o GameMaker pode precisar 
de uma atualização manual.

SOLUÇÃO MANUAL (GameMaker Studio):
----------------------------------

1. ABRIR O GAMEMAKER STUDIO:
   - Abra o projeto "menu de contrucao.yyp"
   - Vá para a aba "Asset Browser" (navegador de recursos)

2. VERIFICAR SE O OBJETO EXISTE:
   - Expanda a pasta "Objects" (Objetos)
   - Procure por "obj_controlador_construcao"
   
3. SE O OBJETO NÃO APARECER:
   - Clique com botão direito na pasta "Objects"
   - Selecione "Create Object" (Criar Objeto)
   - Nomeie como "obj_controlador_construcao"
   - Substitua o arquivo .yy gerado pelo nosso arquivo já criado

4. ADICIONAR EVENTOS:
   - Clique no objeto obj_controlador_construcao
   - Adicione Event → Create
   - Adicione Event → Step → Step
   - Adicione Event → Draw → Draw
   - Cole o código dos arquivos que criamos

5. VERIFICAR REGISTRO:
   - Compile o projeto (F5)
   - Se ainda der erro, feche e reabra o GameMaker Studio
