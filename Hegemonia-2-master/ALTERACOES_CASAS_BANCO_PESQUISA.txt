## 🎮 **COMO USAR O NOVO SISTEMA:**

### **1. CONSTRUIR CASAS:**
```
1. Clique esquerdo na casa para selecionar
2. Casa seguirá o mouse automaticamente
3. Clique direito para posicionar
4. Cada casa adiciona 10 pessoas à cidade (LIMITE MÁXIMO)
5. Cidade tem limite total de 1000 pessoas
```

### **2. CONSTRUIR BANCOS:**
```
1. Clique esquerdo no banco para selecionar
2. Banco seguirá o mouse automaticamente
3. Clique direito para posicionar
4. Banco gera dinheiro baseado na população
```

### **3. ECONOMIA AUTOMÁTICA:**
```
- População: Cada casa = +10 pessoas (LIMITE POR CASA)
- Limite Total: Máximo 1000 pessoas na cidade
- Dinheiro: Cada pessoa = +$25 a cada 3s
- Exemplo: 100 pessoas = $2.500 a cada 3s
- Controle: Sistema previne crescimento infinito
```
