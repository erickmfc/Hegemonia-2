## ⚔️ **SISTEMA SIMPLES DE TIRO E ATAQUE:**

### ✅ **FUNCIONALIDADES IMPLEMENTADAS:**

#### 1. **Sistema de Seleção:**
- **🎯 Clique Esquerdo**: Seleciona a unidade
- **📱 Feedback**: Unidade selecionada fica destacada
- **🔄 Controle**: Apenas uma unidade selecionada por vez

#### 2. **Sistema de Ataque:**
- **⚔️ Clique Direito em Unidade Inimiga**: Ataca a unidade
- **🎯 Tiro Automático**: Cria projétil que segue o alvo
- **💥 Dano**: Aplica 25 de dano por tiro
- **💀 Destruição**: Unidade inimiga é destruída se HP chegar a 0

#### 3. **Sistema de Movimento:**
- **🚶 Clique Direito em Local Vazio**: Move para a posição
- **🎯 Clique Direito em Aliado**: Move para a posição (não ataca)
- **📍 Pathfinding**: Movimento direto para o destino

---

## 🎮 **COMO USAR O SISTEMA:**

### **1. SELECIONAR UNIDADE:**
```
1. Clique esquerdo na unidade desejada
2. Unidade fica selecionada (destaque visual)
3. Sistema está pronto para comandos
```

### **2. ATACAR INIMIGO:**
```
1. Com unidade selecionada, clique direito no inimigo
2. Sistema detecta que é um inimigo
3. Cria tiro automático que segue o alvo
4. Tiro causa dano e pode destruir o inimigo
```

### **3. MOVER UNIDADE:**
```
1. Com unidade selecionada, clique direito em local vazio
2. Sistema detecta que é movimento
3. Unidade se move para a posição
4. Pathfinding automático para o destino
```

---

## 🎯 **DETECÇÃO DE INIMIGOS:**

### **Lógica de Detecção:**
```
1. Clique direito detecta unidade mais próxima
2. Verifica se é diferente da unidade selecionada
3. Verifica se é de nação diferente (inimiga)
4. Se inimiga: Modo Ataque
5. Se amiga ou vazio: Modo Movimento
```

### **Código de Nação:**
```
1 = Jogador (Azul)
2 = Inimigo (Vermelho)
3 = Aliado (Verde)
4 = Neutro (Amarelo)
```
