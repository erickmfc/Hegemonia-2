COMO TESTAR:

1. EXECUTE O JOGO
   - Compile e rode o projeto
   - Agora você inicia com $50000

2. TESTE INICIAL (Tecla T)
   - Pressione T para ver o status do sistema
   - Verifique se todas as variáveis globais estão OK

3. TESTE DO MENU (Tecla B)
   - Pressione B para abrir o menu de pesquisa
   - Confirme que o menu aparece

4. TESTE DE MÚTIPLAS PESQUISAS
   - Clique em vários cartões diferentes
   - Todos devem ficar laranjas agora!
   - Se precisar de mais dinheiro, pressione M

5. TESTE FORÇADO (Tecla R)
   - Pressione R para forçar o início da pesquisa de Borracha
   - Isso testa se o sistema funciona sem cliques

6. ADICIONAR DINHEIRO (Tecla M)
   - Pressione M para adicionar $25000
   - Use quando precisar de mais dinheiro para pesquisas

6. ANÁLISE DOS LOGS:
   Procure por estas mensagens na saída de debug:

   PROCURE POR:
   - "=== CHAVES EM global.nacao_recursos ==="
   - Lista de todas as chaves existentes vs esperadas
   - "AVISO: Chave 'NomeRecurso' não existe em global.nacao_recursos!"
   
   ✅ SUCESSO - Devem aparecer:
   - "=== CLIQUE DETECTADO NO CENTRO DE PESQUISA ==="
   - "Verificando variáveis globais..."
   - "*** CLIQUE DETECTADO NO CARTÃO: [NomeRecurso] ***"
   - "========== PESQUISA INICIADA =========="
   
   ✅ TESTE FORÇADO (Tecla R) - Deve aparecer:
   - "✅ PESQUISA FORÇADA INICIADA COM SUCESSO!"
   - E o cartão de Borracha deve ficar laranja no menu

   ❌ PROBLEMA - Se aparecer:
   - "ERRO CRÍTICO: global.nacao_recursos não é um ds_map válido!"
   - "ERRO CRÍTICO: global.research_timers não é um ds_map válido!"
   - "NENHUM CARTÃO FOI CLICADO"

INFORMAÇÕES TÉCNICAS:
- Timer de pesquisa: 30 segundos (para testes)

TROUBLESHOOTING:
