COMO USAR A NOVA INTERFACE:

1. ATIVAÇÃO DO MODO DE CONSTRUÇÃO
   - Pressione a tecla C para ativar o modo de construção
   - O menu aparece automaticamente na parte inferior da tela
   - Mostra três botões: "CASA", "BANCO" e "QUARTEL"

2. CONSTRUÇÃO DO QUARTEL
   - Clique no botão "QUARTEL" no menu de construção
   - Custo: $800 + 150 minério
   - Menu desaparece automaticamente após seleção
   - Clique no mapa para construir o quartel

3. RECRUTAMENTO DE INFANTARIA
   - Após construir o quartel, clique nele para abrir o menu de recrutamento
   - Interface mostra recursos disponíveis
   - Clique em "RECRUTAR INFANTARIA" para iniciar o treinamento
   - Custo: $100 + 1 população
   - Tempo de treinamento: 5 segundos

4. CONTROLES
   - Tecla C: Ativa/Desativa modo de construção
   - Clique esquerdo: Seleciona construção ou recruta unidades
   - Clique direito: Cancela construção em andamento
   - Clique fora do menu: Fecha interface
